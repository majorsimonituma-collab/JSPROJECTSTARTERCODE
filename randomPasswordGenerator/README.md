# 🔐 Random Password Generator

A simple yet powerful **Random Password Generator** built with Javascript.  
This tool creates secure, customizable passwords to help protect your accounts from brute-force attacks.

---

## ✨ Features
- Generate passwords of any length from 8 - 32 characters.
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Avoid ambiguous characters (optional).
- Fully random and cryptographically secure.
